# SE_Final_Proj

Code based on flask-boilerplate https://github.com/realpython/flask-boilerplate.git

## Starting the webpage
1. Clone the reop
$ git clone https://github.com/jl13148/SE_Final_Proj.git
$ cd SE_Final_Proj/flask-boilerplate

2. Initialize and activate a virtualenv:

$ conda create -n <venv-name>

3. Install the dependencies:
$ pip install -r requirements.txt

4. Run the development server:
$ python app.py

5. Navigate to http://localhost:5000

## Main changes to the boilerplate
1. form.py: LoginForm RegisterForm

2. models.py: User Medication
  
3. app.py: controller

4. /templates/pages/placeholder.home.html: home display
